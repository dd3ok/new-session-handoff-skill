# Expanded Handoff Artifacts

The canonical expanded-mode contract now lives in `references/handoff-contract.md`.

Use expanded mode when a compact `HANDOFF.md` would lose critical recovery context. Keep `HANDOFF.md` as the entry manifest and link focused detail artifacts such as:

- `details/architecture.md`
- `details/changed-files.md`
- `details/validation.md`
- `details/pitfalls.md`
- `details/open-questions.md`

Use the matching `references/detail-*-template.md` file only when a detail artifact needs a durable structure.
